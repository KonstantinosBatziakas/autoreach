Cookie Policy

Last updated: June 2026 · Applies to autoreach.dev and self-hosted AutoReach instances

This Website (autoreach.dev)

The autoreach.dev landing page is a static site hosted on GitHub Pages. It does not use cookies, tracking pixels, or analytics. No personal data is collected when you visit this website.

The only external resource loaded is Google Fonts for typography. Google may set cookies as part of this — see Google's Privacy Policy.

The AutoReach Dashboard (Self-Hosted)

When you run your own AutoReach instance, the dashboard uses a single session cookie:

Cookie Purpose Type Duration
session Keeps you logged into the dashboard Essential Browser session

This cookie is strictly necessary for the dashboard to function. It contains only an encrypted session token — no personal data, no tracking information.

No Analytics or Tracking

AutoReach does not use:

  • Google Analytics or any other analytics service
  • Advertising or retargeting cookies
  • Third-party tracking pixels
  • Social media tracking

Your Control

You can clear cookies at any time through your browser settings. Clearing the session cookie will log you out of the AutoReach dashboard. Since we set no tracking cookies, there is no cookie consent banner — there is nothing to consent to.

For questions, open an issue on GitHub.
