How it works

From zero to sent
in four steps.

01

Find Leads

Enter a city and business type (e.g. "restaurants" in "Athens"). AutoReach queries the Google Maps Places API and saves every result to your leads database.

02

Scrape Emails

Click "Scrape Emails" and AutoReach crawls each business website — homepage, /contact, /about — and extracts whatever email addresses it finds.

03

Generate & Send

Run Outreach and Groq AI writes a tailored cold email for every lead. Sent via Gmail SMTP in seconds — duplicates automatically skipped.

04

Track Results

The dashboard shows your pipeline progress. The Sent Emails log and Report page give you a full history of every contact made.

Start in
60 seconds.

Clone the repo, add your API keys to .env, and launch the dashboard.

$ git clone https://github.com/KonstantinosBatziakas/autoreach
$ cd autoreach
$ python -m venv venv && venv\Scripts\activate
$ pip install flask requests groq python-dotenv beautifulsoup4 schedule
$ python app.py

Requires a free Groq API key, a Gmail app password, and a Google Maps API key.
